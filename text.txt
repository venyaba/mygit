 1 уровень сложности: 1. Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?

  Для клонирования существующего репозитория понадобится ввести git clone <url>
  git clone https://github.com/rep/rep


 2 Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?

 git init - Благодаря этой команде создается структура подкаталога со всеми необходимыми файлами. Кстати, все они расположены в подпапке с названием .git. Пока что проект не находится под контролем учета версий, поскольку в него добавлены только нужные элементы для работы Git.

git add . / git commit -m "some comment" / git remote add origin https://github.com/rep/rep / git push


3 Как получить на локальный компьютер изменения с github.com

 git pull origin master
 From https://github.com/rep/rep

4. Какую информацию нам дают команда :
 git status - отображает состояние рабочего каталога и раздела проиндексированных файлов. С ее помощью можно проверить индексацию 
 изменений и увидеть файлы, которые не отслеживаются Git. 
 git log - команда отображает список последний коммитов



 git remote -v - ключ -v, дает возможность просмотреть адреса для чтения и записи, привязанные к репозиторию
  Это означает, что мы можем легко получить изменения от любого из пользователей. Возможно, что некоторые из репозиториев доступны для 
 записи и в них можно отправлять свои изменения, хотя вывод команды не даёт никакой информации о правах доступа.
